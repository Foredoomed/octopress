# 配置vim
vim /etc/vimrc</p>

<h1 id="section-2">加入下面几行</h1>
<p>syntax on
set nu
set mouse=a
set tabstop=4</p>

<h1 id="ipv6">禁用IPv6</h1>
<p>vim /etc/sysconfig/network
NETWORKING_IPV6=”no”</p>

<h1 id="section-3">设置文件最大打开数</h1>
<p>vim /etc/security/limits.conf
# 加入下面两行
* soft nofile 65535
* hard nofile 65535</p>

<h1 id="section-4">把服务器时间改为上海时间</h1>
<p>cp -f /usr/share/zoneinfo/Asia/Shanghai /etc/localtime</p>

<h1 id="section-5">关闭不需要的服务</h1>
<p>ntsysv  #按空格选择和取消，按F12退出
